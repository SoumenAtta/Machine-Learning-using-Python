Chapter 4 Logistic Regression

Introduction to Logistic Regression¶

Logistic regression is a statistical method used for binary classification tasks, where the outcome variable has only two possible outcomes or classes. It's called "regression" because it predicts the probability of a categorical outcome, and "logistic" because it uses the logistic function to model this probability. Logistic regression is widely used in various fields such as healthcare, marketing, finance, etc., where binary classification is required.

Logistic Function (Sigmoid Function)

The logistic function, also known as the sigmoid function, is a key component of logistic regression. It takes any input value and outputs a value between 0 and 1, which can be interpreted as a probability.

The logistic function is defined as:

$\sigma(z) = \frac{1}{1 + e^{-z}}$

Where:

  • $e$ is the base of the natural logarithm.
  • $z$ is the linear combination of predictor variables and their coefficients.

Logistic Regression Model

The logistic regression model estimates the probability of the occurrence of the binary outcome based on one or more predictor variables. Mathematically, the logistic regression model can be represented as follows:

$P(Y=1|X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1X_1 + \beta_2X_2 + ... + \beta_nX_n)}}$

Where:

  • $P(Y=1|X)$ is the probability of the outcome variable being 1 given the values of predictor variables $X$.
  • $\beta_0, \beta_1, \beta_2, \ldots, \beta_n$ are the coefficients of the model, representing the effects of the predictor variables on the log-odds of the outcome.
  • $X_1, X_2, ..., X_n$ are the predictor variables.

Model Training

The logistic regression model estimates the coefficients $\beta_0, \beta_1, \beta_2, \ldots, \beta_n$ using maximum likelihood estimation. The objective is to maximize the likelihood of observing the given data under the assumed logistic distribution of the outcome variable.

Model Evaluation

After training the model, it needs to be evaluated using a separate dataset (testing data). Common evaluation metrics for classification tasks include accuracy, confusion matrix, precision, recall, and F1-score.

Making Predictions

Once the model is trained and evaluated, it can be used to make predictions on new data. The predicted probabilities can be converted into class labels by applying a threshold (usually 0.5).

Difference between Logistic regression and Linear regression¶

Logistic regression and linear regression are both common techniques used in regression analysis, but they serve different purposes and are applicable in different scenarios. Here's a comparison between logistic regression and linear regression:

1. Purpose:

  • Linear Regression: Linear regression is used for predicting a continuous outcome variable based on one or more predictor variables. It assumes a linear relationship between the predictor variables and the outcome variable.
  • Logistic Regression: Logistic regression is used for predicting a binary outcome variable (0 or 1) based on one or more predictor variables. It models the probability of the outcome variable being in a particular category.

2. Type of Outcome Variable:

  • Linear Regression: The outcome variable in linear regression is continuous. It can take any real value within a given range.
  • Logistic Regression: The outcome variable in logistic regression is categorical (binary). It represents the probability of an event occurring (e.g., yes/no, true/false).

3. Model Representation:

  • Linear Regression: In linear regression, the relationship between the predictor variables and the outcome variable is represented by a linear equation. The predicted values are continuous.
  • Logistic Regression: In logistic regression, the relationship between the predictor variables and the log-odds of the outcome variable is represented by the logistic function (sigmoid function). The predicted values are probabilities between 0 and 1.

4. Assumptions:

  • Linear Regression: Linear regression assumes a linear relationship between the predictor variables and the outcome variable. It also assumes that the errors are normally distributed and have constant variance (homoscedasticity).
  • Logistic Regression: Logistic regression assumes that the log-odds of the outcome variable are linearly related to the predictor variables. It also assumes that the observations are independent and that there is no multicollinearity among the predictor variables.

5. Evaluation Metrics:

  • Linear Regression: Common evaluation metrics for linear regression include mean squared error (MSE), root mean squared error (RMSE), and R-squared.
  • Logistic Regression: Common evaluation metrics for logistic regression include accuracy, precision, recall, F1-score, and area under the ROC curve (AUC-ROC).

6. Implementation:

  • Linear Regression: Linear regression models are often implemented using ordinary least squares (OLS) or gradient descent algorithms.
  • Logistic Regression: Logistic regression models are implemented using maximum likelihood estimation (MLE) or optimization algorithms such as gradient descent.

In summary, while both logistic regression and linear regression are regression techniques, they have different purposes, assumptions, and model representations. Linear regression is suitable for predicting continuous outcomes, while logistic regression is suitable for predicting binary outcomes. The choice between the two depends on the nature of the outcome variable and the problem at hand.

Examples of Simple Logistic Regression and Multiple Logistic Regression¶

Let's discuss logistic regression with one variable (simple logistic regression) and logistic regression with multiple variables (multiple logistic regression), and provide examples for both scenarios:

1. Logistic Regression with One Variable (Simple Logistic Regression):

In simple logistic regression, there is only one predictor variable (X). It's suitable for binary classification tasks where the relationship between the predictor variable and the binary outcome is modeled using the logistic function.

Example:

Let's consider a dataset where we want to predict whether a student passes (1) or fails (0) an exam based on the number of hours they studied.

In [8]:
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

# Generate sample data
np.random.seed(0)
hours_studied = np.random.randint(0, 10, size=100).reshape(-1, 1)  # Predictor variable
pass_exam = np.random.randint(0, 2, size=100)  # Binary outcome variable

# Split data into training and testing sets
X_train, X_test, y_train, y_test = train_test_split(hours_studied, pass_exam, test_size=0.2, random_state=42)

# Initialize and train logistic regression model
logistic_model = LogisticRegression()
logistic_model.fit(X_train, y_train)

# Make predictions
y_pred = logistic_model.predict(X_test)

# Evaluate model
accuracy = accuracy_score(y_test, y_pred)
print("Accuracy:", accuracy)

Accuracy: 0.65

2. Logistic Regression with Multiple Variables (Multiple Logistic Regression):

In multiple logistic regression, there are multiple predictor variables (X1, X2, ..., Xn). It's suitable for binary classification tasks where the relationship between multiple predictor variables and the binary outcome is modeled using the logistic function.

Example:

Let's consider a dataset where we want to predict whether a customer will purchase (1) or not purchase (0) a product based on their age, income, and gender.

In [10]:
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

# Generate sample data
np.random.seed(0)
age = np.random.randint(18, 65, size=100).reshape(-1, 1)  # Predictor variable 1
income = np.random.randint(20000, 100000, size=100).reshape(-1, 1)  # Predictor variable 2
gender = np.random.randint(0, 2, size=100).reshape(-1, 1)  # Predictor variable 3 (0: Female, 1: Male)
purchase = np.random.randint(0, 2, size=100)  # Binary outcome variable

# Combine predictor variables into one feature matrix
X = np.concatenate((age, income, gender), axis=1)

# Split data into training and testing sets
X_train, X_test, y_train, y_test = train_test_split(X, purchase, test_size=0.2, random_state=42)

# Initialize and train logistic regression model
logistic_model = LogisticRegression()
logistic_model.fit(X_train, y_train)

# Make predictions
y_pred = logistic_model.predict(X_test)

# Evaluate model
accuracy = accuracy_score(y_test, y_pred)
print("Accuracy:", accuracy)

Accuracy: 0.45

In both examples, we use scikit-learn's LogisticRegression class for model training and prediction. The accuracy of the model is evaluated using the accuracy_score function from scikit-learn.metrics.
